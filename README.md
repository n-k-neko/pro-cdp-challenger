## ProCDP Challengerについて
**ProCDP Challenger**は、資格試験学習のモチベーション維持をサポートするためのアプリケーションです。主な機能は以下の通りです。

- **受験予定の資格試験を登録**: 試験日や目標学習時間を管理。
- **学習時間の記録**: 資格試験に向けた日々の学習時間を記録し、進捗状況を可視化。
- **フォロー機能**：他ユーザーをフォロー。

## リリーススケジュール
概ね2週間を1スプリントとして機能をリリースします。
